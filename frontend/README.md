# Frontend

Angular 22 app for MangaTracker.

## Commands

```bash
npm start                                      # dev server on localhost:4200
npm run build                                  # production build
npm test                                       # Vitest unit tests
npm run lint                                   # ESLint
npm run format:check                           # Prettier check
npm run e2e                                    # mocked Playwright flow
npm run e2e:integration                        # real backend integration flow
```
